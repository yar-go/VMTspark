# ФУОЗ для МТ Дніпро/Урал

ФУОЗ на мікроконтролері Attiny13 для радянських важких мотоциклів. В розробці!

## Особливості

- Робота з датчиком Холла та комутатором від ВАЗ 2108.
- Незалежність від форми модулятора.
- Різні додаткові функції

## Розробка

Розробка виконується повільно, оскільки поки нема можливості перевірити в реальних
умовах. Через що пропадає мотивація закінчити прошивку до кінця.
